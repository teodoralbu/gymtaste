For each interactive div, add role, tabIndex, and onKeyDown attributes to make them keyboard-accessible per UX-01.

**src/components/user/AvatarUpload.tsx (line ~84):**
Find the `<div>` with `onClick={() => fileInputRef.current?.click()}`. Add:
```tsx
<div
  role="button"
  tabIndex={0}
  onClick={() => fileInputRef.current?.click()}
  onKeyDown={(e) => {
    if (e.key === 'Enter' || e.key === ' ') {
      e.preventDefault()
      fileInputRef.current?.click()
    }
  }}
  aria-label="Upload avatar photo"
  style={{ cursor: 'pointer', /* ...existing styles */ }}
>
```

**src/components/rating/ReviewCard.tsx (line ~40):**
Find the `<div>` with `onClick={() => setExpanded(!expanded)}`. Add:
```tsx
<div
  role="button"
  tabIndex={0}
  onClick={() => setExpanded(!expanded)}
  onKeyDown={(e) => {
    if (e.key === 'Enter' || e.key === ' ') {
      e.preventDefault()
      setExpanded(!expanded)
    }
  }}
  aria-expanded={expanded}
  aria-label={expanded ? "Collapse review" : "Expand review"}
  /* ...existing styles and props */
>
```

**src/components/rating/CommentsSection.tsx (line ~116):**
The backdrop overlay `<div onClick={onClose}>` is an overlay dismiss pattern. Convert to:
```tsx
<div
  role="button"
  tabIndex={-1}
  onClick={onClose}
  onKeyDown={(e) => {
    if (e.key === 'Escape') onClose()
  }}
  aria-label="Close comments"
  /* ...existing styles */
>
```
Note: tabIndex={-1} because backdrop should not be in tab order — Escape key handles keyboard dismiss.

**src/components/ui/Modal.tsx (line ~32):**
