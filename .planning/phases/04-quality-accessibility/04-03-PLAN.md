**Part C — AvatarUpload keyboard accessibility (UX-01):**

In `src/components/user/AvatarUpload.tsx`, find the `<div>` with `onClick={() => fileInputRef.current?.click()}` (line ~84). Add these attributes:
```tsx
<div
  role="button"
  tabIndex={0}
  onClick={() => fileInputRef.current?.click()}
  onKeyDown={(e) => {
    if (e.key === 'Enter' || e.key === ' ') {
      e.preventDefault()
      fileInputRef.current?.click()
    }
  }}
  aria-label="Upload avatar photo"
  style={{ cursor: 'pointer', /* ...existing styles */ }}
>
```
