# Phase 12 Plan 01: Profile Body Stats & Dosage Calculator

## Objective

Add body stats fields (height, weight, fitness goal) to user profiles, and display personalized supplement dosage recommendations on product pages with safety disclaimers.

## Context

- Settings page (`src/app/settings/page.tsx`) has avatar, profile, badge, account sections
- User type in `src/lib/types.ts` needs height_cm, weight_kg, fitness_goal fields
- Product page (`src/app/products/[slug]/page.tsx`) displays product specs, nutrition, label
- Products have `caffeine_mg`, `citrulline_g`, `beta_alanine_g` fields for dosage calculations
- Auth context provides `profile` with user data and `refreshProfile` to reload
- Constraint: Must cite safe dosing ranges, not prescribe medical advice

## Tasks

### Task 1: Add body stats fields to User type and settings form
type="auto"

**Do:**
1. Add `height_cm`, `weight_kg`, `fitness_goal` to User interface in `src/lib/types.ts`
2. Update Database type to include new fields (nullable, optional on insert)
3. Add `FITNESS_GOALS` constant to `src/lib/constants.ts`
4. Add body stats section to settings page with height (cm), weight (kg), and fitness goal selector
5. Save body stats to Supabase on form submission (reuse existing save handler pattern)

**Done when:** Settings page shows height, weight, and goal fields that save to database.

### Task 2: Create dosage calculator component for product pages
type="auto"

**Do:**
1. Create `src/lib/dosage.ts` with dosage calculation logic:
   - Caffeine: 3-6mg/kg body weight (ISSN recommendation)
   - Citrulline: 6-8g/day (ISSN recommendation)
   - Beta-Alanine: 3.2-6.4g/day (ISSN recommendation)
   - Adjust ranges based on fitness goal (endurance = higher caffeine ceiling, muscle gain = higher citrulline)
2. Create `src/app/products/[slug]/DosageCalculator.tsx` client component:
   - Reads user profile from auth context
   - Shows personalized dosage range for each ingredient the product has
   - Shows how many scoops/servings to take based on product's per-serving amounts
   - Non-dismissable safety disclaimer at the bottom
   - If user not logged in or no body stats: show prompt to set up profile
3. Add DosageCalculator to product page between nutrition and label sections

**Done when:** Product page shows personalized dosage recommendations with safety disclaimer.

## Verification

- Settings page renders body stats fields
- Saving body stats persists to database
- Product page shows dosage calculator when user has body stats
- Product page shows "set up profile" prompt when user has no body stats
- Safety disclaimer is always visible and non-dismissable

## Success Criteria

- CALC-01: User can enter height, weight, and fitness goal on profile settings
- CALC-02: Product pages show dosage recommendation based on user profile
- CALC-03: Calculator includes safety disclaimer citing ISSN recommended safe ranges
