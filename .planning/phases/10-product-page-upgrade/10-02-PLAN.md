**Create `src/app/products/[slug]/NutritionSwitcher.tsx`:**

A `'use client'` component that receives product nutritional data as props and renders a segmented tab control to switch between per-scoop, per-serving, and per-100g views.

Props interface:
```typescript
interface NutritionSwitcherProps {
  calories: number | null
  proteinG: number | null
  carbsG: number | null
  fatG: number | null
  sugarG: number | null
  sodiumMg: number | null
  scoopWeightG: number | null
  servingWeightG: number | null
}
```

Implementation details:
- Use `useState<'scoop' | 'serving' | '100g'>('serving')` for the active tab
- Compute available units: always include `'serving'`. Add `'scoop'` (first position) only if BOTH `scoopWeightG` and `servingWeightG` are non-null and non-zero. Add `'100g'` (last position) only if `servingWeightG` is non-null and non-zero.
- `getMultiplier` function: `'serving'` returns 1, `'scoop'` returns `scoopWeightG / servingWeightG`, `'100g'` returns `100 / servingWeightG`. Guard against division by zero (return 1 as fallback).
- Build a nutrients array: `[{ label: 'Calories', value: calories, unit: '', decimals: 0 }, { label: 'Protein', value: proteinG, unit: 'g', decimals: 1 }, { label: 'Carbs', value: carbsG, unit: 'g', decimals: 1 }, { label: 'Fat', value: fatG, unit: 'g', decimals: 1 }, { label: 'Sugar', value: sugarG, unit: 'g', decimals: 1 }, { label: 'Sodium', value: sodiumMg, unit: 'mg', decimals: 0 }]`. Filter out items where `value` is null.
- If no nutrients have data (all null), render nothing (return `null`).
- Render: section heading "Nutrition Facts", the `m-segment` tab bar (buttons with `m-segment-tab` class, `active` class on selected), then a CSS grid of nutrient cards (3 columns on wider screens, 2 on narrow). Each card shows the computed value (value * multiplier, toFixed with correct decimals) + unit in large bold text, label below in small muted text.
- Use inline styles consistent with project patterns. Card style: `backgroundColor: 'var(--bg-card)', border: '1px solid var(--border-soft)', borderRadius: 'var(--radius-md)', padding: '16px', textAlign: 'center'`. Value style: `fontSize: '20px', fontWeight: 800, color: 'var(--text)'`. Label style: `fontSize: '11px', color: 'var(--text-faint)', textTransform: 'uppercase', letterSpacing: '0.06em', fontWeight: 600, marginTop: '4px'`.
- Tab labels: `'Per scoop'`, `'Per serving'`, `'Per 100g'`.
